Calculators

XP Level → Total XP

Uses the Java Edition XP formulas from the Minecraft Wiki. Enter a level and see how much total XP you need from level 0 to reach it.

Level 30 requires 1395 total XP.

Behind the scenes, the formula changes for 0–16, 17–31 and 32+ levels, matching vanilla behavior.
